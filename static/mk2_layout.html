MK2 — NI-faithful VJ layout v2 (manual-aligned, click pages)

All silkscreen labels per NI Maschine MK2 Hardware Control Reference v1.8. Each zone shows NI intent → VJ remap. Click any top-row PAGE button to preview the swap.

Key finding: NI explicitly left the 8 buttons above the OLEDs unlabeled by design — they're the only unlabeled elements on the device, intended to be repurposed per mode. Mapping them to VJ page selectors is the single most NI-faithful repurposing available. The rest of this layout keeps NI silkscreens as cues so muscle memory stays intact.

NI intent: Unlabeled, context-dependent (page tabs for the current mode).  → VJ remap: 8 pages × 7 verticals = 56 folder slots. Active page highlighted.

Status of unknowns: bits 8-17 are unmapped in our driver. Per the NI manual research, those bits almost certainly cover the CONTROL / STEP / BROWSE / SAMPLING / ALL / AUTO WRITE column AND the bottom SHIFT / NOTE REPEAT / ERASE / TAP / REC cluster. One 10-min discovery session per cluster pins them all. Then each gets a VJ remap that respects NI's silkscreen intent (e.g. BROWSE = open library nav, ERASE = clear pad/clip, REC = start session recording).
